## Project Overview

Comprehensive HR analytics solution replacing manual spreadsheet processes with automated, refresh-ready reporting.

### Report Modules

- **Headcount Dashboard**: Real-time employee count by department, location, level
- **Turnover Analysis**: Voluntary/involuntary splits with tenure breakdowns
- **Diversity Metrics**: Gender, ethnicity, age distribution tracking
- **Compensation Analysis**: Pay equity analysis and market benchmarking

### Automation Features

- One-click data refresh from HRIS exports
- Automatic chart and pivot table updates
- Email distribution via Outlook integration
- Change highlighting for month-over-month comparison

### Data Transformations

Power Query handles:
- Data cleansing and standardization
- Employee status calculations
- Tenure and age group binning
- Org hierarchy flattening

### Outcomes

- Eliminated 20 hours/month of manual reporting
- Provided executives with weekly HR metrics
- Supported DEI initiatives with accurate diversity tracking
