## Project Overview

Enterprise-grade 3-statement financial model supporting strategic planning and annual budgeting processes.

### Model Components

- **Income Statement**: Monthly P&L with 24-month rolling forecast
- **Balance Sheet**: Automated working capital calculations
- **Cash Flow Statement**: Direct and indirect method support
- **Scenario Manager**: Best/Base/Worst case modeling

### Advanced Features

- Dynamic date ranges with automatic period detection
- Sensitivity analysis tornado charts
- Monte Carlo simulation for risk assessment
- Variance commentary templates

### Technical Highlights

- 50+ Power Query connections for data refresh
- VBA automation for report generation
- Protected inputs with validation rules
- Executive summary dashboard

### Impact

- Reduced budget cycle from 8 weeks to 3 weeks
- Enabled real-time scenario planning during board meetings
- Standardized financial modeling across 12 business units
