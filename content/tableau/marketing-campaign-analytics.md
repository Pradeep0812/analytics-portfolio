## Project Overview

Comprehensive marketing analytics solution providing visibility into campaign performance across all channels.

### Dashboard Features

- **Channel Performance**: Compare ROI across email, social, paid search, and display
- **Attribution Modeling**: First-touch, last-touch, and data-driven attribution comparison
- **Funnel Analysis**: Visualize conversion funnels from awareness to purchase
- **A/B Test Results**: Statistical significance testing for campaign experiments

### Data Sources

- Google Analytics 4
- Facebook Ads API
- Google Ads API
- Salesforce CRM
- Custom ETL pipeline

### Key Insights Delivered

- Identified 40% budget waste in underperforming channels
- Discovered cross-channel synergies worth $500K annually
- Reduced CAC by 25% through optimized targeting
